# 串口调试工具 流程图

## 1. 程序启动流程

```
开始
  │
  ▼
main()
  │
  ├─► 创建 QApplication
  │
  ├─► 设置 Fusion 风格
  │
  ├─► 加载应用图标
  │
  └─► 创建并显示 SerialTool 窗口
        │
        ├─► 初始化变量
        │     • 串口相关 (serial_thread, reconnect_timer 等)
        │     • 缓冲区设置
        │     • 主题/字体设置
        │     • 搜索相关变量
        │
        ├─► setup_icon() - 加载窗口图标
        │
        ├─► setup_ui() - 创建UI
        │     │
        │     ├─► apply_theme() - 应用主题
        │     │
        │     ├─► setup_toolbar() - 工具栏
        │     │     • 显示/隐藏配置
        │     │     • 连接/断开
        │     │     • 清除接收
        │     │     • 保存接收
        │     │
        │     ├─► create_receive_frame() - 接收区域
        │     │     │
        │     │     ├─► create_control_frame() - 控制栏
        │     │     │     • 复选框：时间戳/十六进制/暂停/行号/自动滚屏
        │     │     │     • 搜索栏：输入框/大小写/上/下按钮
        │     │     │     • 统计：接收/发送/缓冲区/状态
        │     │     │
        │     │     └─► CustomTextBrowser + LogHighlighter
        │     │
        │     └─► create_config_frame() - 配置区域
        │           │
        │           ├─► create_config_group() - 串口配置
        │           │     • 端口选择
        │           │     • 波特率/数据位/停止位/校验
        │           │     • 连接按钮
        │           │
        │           └─► create_send_group() - 发送区域
        │                 • 发送文本框
        │                 • 发送选项
        │
        ├─► setup_menu() - 菜单栏
        │     • 视图：主题/字体
        │     • 编辑：查找/上一个/下一个
        │     • 设置：缓冲区设置
        │     • 帮助：关于
        │
        └─► refresh_ports() + 启动刷新定时器
              │
              └─ 每2秒刷新一次串口列表
```

## 2. 串口连接流程

```
用户点击"连接"按钮
  │
  ▼
toggle_connection()
  │
  ├─ 串口线程是否运行中?
  │     │
  │     ├─ 是 ──► disconnect_serial()
  │     │           │
  │     │           ├─► 停止线程
  │     │           ├─► 停止定时器
  │     │           └─► 更新UI
  │     │
  │     └─ 否 ──► connect_serial()
  │                 │
  │                 ├─► 检查端口选择
  │                 │
  │                 ├─► 获取串口参数
  │                 │     • 波特率/数据位/停止位/校验
  │                 │
  │                 ├─► 创建 SerialThread
  │                 │
  │                 ├─► 连接信号
  │                 │     • data_received
  │                 │     • error_occurred
  │                 │
  │                 ├─► 启动线程
  │                 │
  │                 └─► 更新UI
  │                       • 按钮→"断开"
  │                       • 启用发送
  │                       • 状态显示
```

## 3. 数据接收流程

```
SerialThread 工作线程
  │
  ▼
循环: 读取串口数据
  │
  ├─ 有数据 ──► 发出 data_received 信号
  │                  │
  │                  └─► on_data_received(data)
  │                        │
  │                        ├─► 暂停显示? ──► 是 ──► 返回
  │                        │
  │                        ├─► 缓冲区管理
  │                        │     • 超限→丢弃旧数据
  │                        │
  │                        ├─► 更新接收计数
  │                        │
  │                        ├─► 格式化数据
  │                        │     • 十六进制 或 文本(UTF-8)
  │                        │
  │                        ├─► 添加时间戳(如开启)
  │                        │
  │                        ├─► 插入显示区域
  │                        │
  │                        ├─► 保持搜索高亮
  │                        │
  │                        └─► 自动滚屏(如开启)
  │
  └─ 无数据 ──► 短暂休眠后继续
```

## 4. 数据发送流程

```
用户点击"发送"按钮
  │
  ▼
send_data()
  │
  ├─ 串口未连接? ──► 显示警告，返回
  │
  ├─ 获取发送文本
  │
  ├─ 缓冲区满? ──► 显示警告，返回
  │
  ├─ 模式选择
  │     │
  │     ├─ 十六进制 ──► 解析Hex字符串
  │     │
  │     └─ 文本 ──► UTF-8编码
  │
  ├─ 添加新行(如开启)
  │
  └─► 写入串口
        │
        └─► 更新发送计数
```

## 6. 主题切换流程

```
用户选择主题

  ▼

  │
  ├─► 更新选中状态
  │
  └─► apply_theme(theme)
        │
        ├─► 设置调色板
        │

        │

        │

```

## 7. 窗口关闭流程

```
用户关闭窗口
  │
  ▼
closeEvent()
  │
  ├─ 串口连接中?
  │     │
  │     ├─ 是 ──► 弹出确认对话框
  │     │           │
  │     │           ├─ 确认 ──► 断开连接 → 退出
  │     │           │
  │     │           └─ 取消 ──► 不退出
  │     │
  │     └─ 否 ──► 直接退出
  │
  └─ 程序结束
```

## 9. 功能模块一览

| 模块 | 功能描述 |
|------|----------|
| **串口配置** | 端口选择、波特率(支持自定义)、数据位、停止位、校验位 |
| **数据接收** | 实时显示、十六进制/文本切换、时间戳、缓冲区管理 |
| **数据发送** | 文本/十六进制、重复发送、新行添加 |
| **搜索功能** | 查找、高亮、上下导航、大小写敏感、结果计数 |
| **主题切换** | 暗黑/浅色主题、实时切换 |
| **字体设置** | 5种字体名称、8种字号 |
| **数据保存** | 导出接收数据到TXT文件 |
| **菜单功能** | 视图、编辑、设置、关于 |
